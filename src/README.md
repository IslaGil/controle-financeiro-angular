# 💰 Controle Financeiro

Sistema web desenvolvido em Angular para gerenciamento de receitas e despesas.

## 🚀 Funcionalidades
- Login com validação
- Proteção de rotas (AuthGuard)
- Cadastro de receitas e despesas
- Cálculo automático de saldo
- Persistência com localStorage
- Exclusão de transações

## 🛠️ Tecnologias
- Angular
- TypeScript
- SCSS

## ▶️ Como rodar
```bash
npm install
ng serve
